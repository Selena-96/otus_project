# otus_project
Проект для курса "1С Архитектор"

Для проекта использована БСП версии 3.0.3.341. Работа велась в конфигураторе, все добавленные объекты имеют префикс "otus_".
Также все новые объекты включены в подсистему otus_Интеграция:

Все диаграммы находятся в папке doc/Схемы, где это возможно добавлены файлы puml, также все результаты в подпапке Скриншоты.

# YAxUnit
В расширении YAxUnit 3 общих модуля, модуль ОМ_СозданиеОбъектов нужен только для подготовки тестовых данных для двух других.

# Vanessa Automation
Сценарии тестирования находятся в отдельной папке doc/Сценарии тестирования. Фича-файл "00_01 ПодготовкаДанных" также служит для подготовки данных к корректному выполнению сценариев.
